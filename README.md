## Objectif

Ce projet automatise la récupération et l'analyse des données financières des entreprises du CAC 40 via l'API Yahoo Finance. Il produit un fichier Excel structuré prêt à être connecté à Power BI pour la visualisation.

Compétences démontrées : Python · Analyse financière · Automatisation · Power BI · Excel



## Installation

1. Installer les librairies : pip install -r requirements.txt
2. Lancer le script : python cac40_analyse.py



## Ce que le script produit

- Onglet 1 : 20 entreprises du CAC 40 avec 16 indicateurs financiers
- Onglet 2 : Moyennes par secteur (Luxe, Banque, Énergie...)
- Onglet 3 : Top 5 entreprises par ratio (marge, levier, EV/EBITDA)
